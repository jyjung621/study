

## Express session

- 정보 출처
    > https://github.com/expressjs/session
- 설치
    ```
    npm install -s express-session
    ```

 - NODEMON
    > pm2와 비슷하게 파일이 수정되면 reload해주는것
    ```
    nodemon express-session.js
    ```


#### session-file-store

 - 설치
    ```
    npm install session-file-store
    ```
